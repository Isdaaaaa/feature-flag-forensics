# Roadmap

## Phase 1 — Foundations
- Set up Next.js + TypeScript + Tailwind scaffold
- Add linting, formatting, and basic layout shell
- Commit sample fixture structure

## Phase 2 — Data + Engine
- Implement fixture ingestion for flag configs, user attributes, and session events
- Build deterministic rules evaluation with trace outputs
- Expose trace data via API routes or server actions

## Phase 3 — Explanations
- Create explanation pane with why/why-not reasoning per rule
- Render timeline of flag decisions tied to events
- Add conflict/stale flag detection heuristics

## Phase 4 — Visualization
- Graph view of flag dependencies, rollout splits, and conflicts
- Highlight rule firing order and edge cases visually

## Phase 5 — Support + Demo
- Export/shareable session report for support engineers
- Seed demo workspace with guided scenario
- Capture screenshots/GIFs for portfolio showcase
